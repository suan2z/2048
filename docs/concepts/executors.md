# 안 2 — 심판의 집행자들

> **세계의 균형을 유지하는 비밀 조직의 여성 집행자들이 균열된 현실을 봉합하는 다크 판타지 RPG**

---

| 항목 | 내용 |
|------|------|
| 가제 | 림버스 카제나 : 심판의 집행자들 |

## 세계관

현실 세계와 평행하게 존재하는 **심연층(深淵層)** — 억눌린 감정, 집착, 공포가 응집되어 생겨난 이형(異形)의 괴이가 현실로 흘러나오기 시작했다.

이를 막기 위해 설립된 비밀 조직 **카제나(KAZENA)**. 플레이어는 조직의 신임 지휘관으로서 집행자들을 편성해 사건을 처리한다.

각 챕터는 **독립된 사건 파일** 형식으로 전개되며, 챕터가 쌓일수록 조직 내부의 거대한 음모가 드러난다.

> *"균열은 닫히지 않습니다. 다음 사건은 이미 시작됐어요."*

---

## 핵심 특장점

### AI 리소스 활용
- 집행자별 고유 복장 및 무기 일러스트 (AI 생성)
- 현대+판타지 혼합 설정 → 다양한 캐릭터 비주얼 표현 가능
- 각 사건 해결 후 **집행 보고서 더빙 씬** 으로 성우 활용 극대화

### 사건 파일 구조
```
챕터 구성
├── 브리핑        : 사건 개요 + 집행자 편성
├── 현장 탐색     : 탐색 파트 (NPC 대화, 단서 수집)
├── 심연층 진입   : 턴제 전투 파트
└── 집행 완료     : 결말 + 숨겨진 진실 조각 공개
```

## 캐릭터 구조

| 직책 | 설명 | 전투 역할 |
|------|------|-----------|
| 검사관 | 냉철한 판단력, 규칙을 절대시 | 단일 딜 / 디버프 |
| 봉인사 | 이형을 봉인하는 능력자, 감정적 | 지원 / 봉인 특화 |
| 추적자 | 심연층에서 태어난 존재, 정체 불명 | 광역 / 패시브 강화 |
| 조율사 | 조직과 현실 사이를 조율하는 외교관 | 버프 / 파티 시너지 |
| 미확인 | DLC 추가 예정 | - |

---

## 전투 시스템

- **속성 상성** : 괴이 유형에 따라 유효 집행자 조합이 달라짐
- **진형 배치** : 전방 / 후방 포지션에 따른 전략 변수
- **시너지 발동** : 집행자 조합에 따른 특수 연계기 발동
- **심연 오염도** : 전투가 길어질수록 집행자의 이성이 흔들리는 리스크 요소

## 강점 / 약점

| 강점 | 약점 |
|------|------|
| 챕터 단위 구조 → DLC 확장에 최적 | 두 참고작의 중간 → 독창성 약할 수 있음 |

| 사건 해결 구조 → 플레이어 몰입도 높음 | 세계관 입문 허들 존재 |
