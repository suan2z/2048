## 세계관

미래 디스토피아. 국가가 운영하는 비밀 시설 **카제나(KAZENA)** — 범죄자, 이단자, 그리고 체제에 불편한 존재들이 기억을 강제로 지운 채 수용되는 곳.

플레이어는 카제나에 새로 부임한 **관리인**으로서 수감자들을 이끌며 임무를 수행하고, 시설 깊숙이 숨겨진 진실에 점차 접근해간다.

> *"당신이 여기 있는 건 실수가 아닙니다. 당신도 결국 잊게 될 것입니다."*

---

## 핵심 특장점

### AI 리소스 활용
- 고퀄리티 여성 캐릭터 일러스트 (AI 생성)
- 각 캐릭터의 **기억 회복 씬**을 전문 성우 풀 더빙으로 연출
- 시나리오 집필에 집중 → 캐릭터 서사가 게임의 핵심 감정선

### 기억 회복 서사 구조
각 여성 캐릭터는 지워진 과거를 가지며, 스토리 진행에 따라 파편화된 기억을 회복한다.

```
기억 회복 3단계
├── 1단계 : 단편 기억 (전투 중 플래시백)
├── 2단계 : 기억 조각 수집 (수집 요소)
└── 3단계 : 완전 회복 (챕터 클라이맥스 풀 더빙 씬)
```

## 캐릭터 구조

| 캐릭터 유형 | 설명 | 스킬 계열 |
|------------|------|-----------|
| 방화범 | 분노를 억누른 여성, 기억 속에 거대한 화재 | 화염 / 광역 공격 |
| 사기꾼 | 모두를 믿지 않는 냉소적 인물 | 디버프 / 기만 / 도주 |
| 암살자 | 감정 없이 명령을 따랐던 과거 | 단일 특화 / 치명타 |
| 폭동 주동자 | 체제에 맞선 혁명가의 잔재 | 버프 / 아군 강화 |
| 미확인 | DLC 추가 예정 | - |

---

## 전투 시스템

- **죄목 기반 스킬** : 각 캐릭터의 수감 이유에서 파생된 고유 기술 체계
- **정신력 게이지** : 기억 회복 이벤트에 따라 전투력 상승 / 붕괴 분기
- **감옥 층 구조** : 층마다 다른 규칙과 환경 조건
