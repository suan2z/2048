# 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 턴제 카드 RPG |
| 플랫폼 | Steam (패키지) |
| 예상 플레이타임 | 1회차 50~60시간 |
| 파티 구성 | 최대 3인 출전 (보유 캐릭터 3인 이상 가능) |
| 전투 방식 | 제한된 슬라이드 횟수 안에 핸드 카드 숫자 타일을 보드에 준비 → 액션 버튼으로 일괄 실행 → 핸드 교체 → 적 행동 |
| 유사 게임 | Slay the Spire (카드), Puzzle Quest (퍼즐+RPG), Into the Breach (행동 예고+대응) |

---

## 논의사항

- 현재 적의 턴에 대한 정의가 없음. 3턴 행동일 경우, 말그대로 3턴이 다 지나가면 행동하게됨.
- 기존방식대로 슬라이딩 횟수로 적의 턴을 정의하고, 슬라이딩 소모로 인하여 적의 행동 턴이 0이 되었을때 적 턴에서 해당 행동을 실행하는 방식이어도 괜찮지 않을까.

---

## 핵심 컨셉

**제한된 슬라이드 안에서 핸드를 얼마나 완성할 것인가.**

- 매 턴 시작 시 핸드에 카드가 드로우되며, 카드마다 **발동에 필요한 숫자 타일**이 지정됨
- 플레이어는 제한된 슬라이드 횟수 안에 보드를 조작해 핸드 카드에 맞는 타일을 준비
- **액션 버튼**을 누르면 보드에 타일이 준비된 카드만 실행되고, 핸드 전체를 버린 뒤 새로 드로우
- 액션 버튼을 누를 때마다 적 행동 카운트가 1 감소하며, 카운트 0 도달 시 예고 행동 실행
- **높은 숫자일수록 강력한 카드** — 그러나 높은 숫자를 만들려면 슬라이드가 더 필요

핵심 딜레마:
> *"핸드에 64짜리 필살 카드가 있다. 만들려면 슬라이드가 5번은 더 필요한데 남은 횟수가 3번밖에 없다. 64를 포기하고 4·8·16만 확실히 실행할까, 아니면 도박을 걸까?"*

---

## 코어 루프

```
스토리 챕터 진행
    ↓
전투 돌입 — 보드 초기화 + 핸드 드로우
    ↓
보드 슬라이드 (턴당 횟수 제한, 적 카운트에 영향 없음)
    ↓
핸드 카드 숫자에 맞는 타일 준비 → 액션 버튼
    ↓
타일 있는 카드 실행 / 없는 카드는 버려짐 → 핸드 전체 교체
    ↓
모든 적 카운트 -1 → 카운트 0인 적 행동 실행
    ↓
[반복] 적 전멸 → 전투 승리 → 카드 보상 획득
    ↓
거점 귀환 → 덱 편집 / 캐릭터 강화
    ↓
다음 챕터 진입
```

## 전투 시스템

### 보드 슬라이드와 액션 버튼

```
[ 에너지 보드 ]          [ 적 행동 카운트 ]

┌────┬────┬────┬────┐    고블린 검사   ▶ 3
│  2 │    │  4 │    │    오크 마법사   ▶ 1  ← 다음 턴 공격!
├────┼────┼────┼────┤    보스 드래곤   ▶ 5
│    │  2 │    │  4 │
├────┼────┼────┼────┤
│  4 │    │  2 │    │
├────┼────┼────┼────┤
│    │  4 │    │  2 │
└────┴────┴────┴────┘
```

- 플레이어는 턴당 정해진 횟수 이내에서 보드를 슬라이드하여 타일을 준비. 슬라이드는 적 카운트에 영향 없음
- **액션 버튼**을 누르면 핸드의 카드 중 보드에 해당 타일이 있는 카드만 실행됨
- 액션 버튼 이후 핸드 전체가 소모덱으로 이동하고 새로 드로우, 그다음 **모든 적 카운트 -1**
- 카운트가 0인 적은 **즉시 예고된 행동 실행** 후 카운트 리셋

### 스킬 카드와 발동 조건

- 출전 캐릭터의 스킬 카드를 합산한 덱에서 매 전투 초기 **5장 드로우** — 핸드덱 구성
- 각 카드에는 **발동 숫자**가 표시됨

| 발동 숫자 | 스킬 등급 | 예시 |
|:---------:|:---------:|------|
| 4 | 기본 | 단일 약공격, 소량 방어막 |
| 8 | 일반 | 단일 공격, 디버프 1개 |
| 16 | 강화 | 중간 공격, 광역 디버프, 아군 회복 |
| 32 | 강력 | 강한 광역 공격, 파티 전체 방어막 |
| 64 | 필살 | 캐릭터 전용 필살기 |
| 128 | 초월 | 전투 판도를 뒤집는 초강력 효과 |

- 보드에 해당 숫자 타일이 존재하면 **타일 소모 → 카드 발동**
- 같은 숫자로 발동 가능한 카드가 여러 장이면 원하는 카드를 선택

### 적 행동 예고

- 모든 적은 항상 **다음 행동**을 공개 표시
  - 예: "오크 마법사 — 카운트 1 — 전체 공격 12 예고"
- 플레이어는 이 예고를 보고 남은 슬라이드 횟수와 타일 상황을 계산해 우선순위 결정

### 핵심 긴장 구조

```
높은 숫자 타일을 욕심낼수록
    → 강력한 카드 발동 가능
    → 그러나 슬라이드를 더 소모 → 낮은 카드들도 못 만들 리스크

낮은 카드들만 확실히 챙길수록
    → 실행 보장, 슬라이드 절약
    → 그러나 화력이 낮아짐 → 전투가 길어짐
```

### 보드 위기 상황

- 보드가 꽉 차면 슬라이드 시 새 타일이 생성되지 않음
- 슬라이드해도 타일이 이동하지 않으면 **행동 불능** — 카드 발동도 슬라이드도 막힘
- 보드 관리 실패 = 전략 붕괴의 직접 원인

### 파티 연계 발동

- 같은 턴(슬라이드 없이 연속)에 서로 다른 캐릭터의 카드를 연속 발동하면 **연계 보너스**
  - 예: A 캐릭터 16 카드 → B 캐릭터 16 카드 연속 발동 → 추가 광역 피해
- 연계는 타일 2개를 한 번에 소모하므로 보드 계획이 필요

## 성장 시스템

### 덱 성장 (챕터 내 — 챕터 클리어 시 리셋)

- 전투 보상: 카드 3장 중 1장 선택 → 덱에 추가
- 카드 강화: 같은 카드 2장 합성 → 발동 숫자 감소 (예: 16 → 12) 또는 효과 강화
- 챕터 클리어 시 획득한 카드 전부 초기화

### 챕터 패시브 (챕터 내 — 챕터 클리어 시 리셋)

챕터 진행 중 레벨업 시 보드·전투 관련 패시브 선택. 선택한 패시브는 챕터 내 이후 전투에 즉시 적용. 챕터 클리어 시 초기화.

| 패시브 | 효과 |
|--------|------|
| 선제 타일 | 전투 시작 시 보드에 4 타일 2개 추가 생성 |
| 타일 절약 | 카드 발동 후 소모 타일 수치의 절반을 보드에 반환 |
| 카운트 지연 | 전투 시작 시 모든 적 카운트 +1 |
| 연계 전문 | 연계 발동 보너스 피해 +50% |

### 유물 (챕터 내 — 챕터 클리어 시 리셋)

챕터 보스 처치 시 유물 획득 → 보드 규칙 또는 카드 조건을 변경. 챕터 클리어 시 초기화.

| 유물 | 효과 |
|------|------|
| 시간의 모래 | 특정 카드 발동 시 모든 적 카운트 +1 |
| 황금 나침반 | 슬라이드 3회마다 원하는 위치에 타일 1개 지정 생성 |
| 공명석 | 64 이상 타일 소모 시 추가로 핸드덱 1장 드로우 |

### 유저 레벨 (영구 성장)

- 챕터 플레이로 경험치를 획득하며, 유저 레벨이 상승하면 챕터에서 등장하는 유물의 종류가 영구적으로 확장된다.
- 승패 여부와 무관하게 경험치 지급.

### 장비 (영구 성장)

- 아웃게임 상점 또는 뽑기를 통해 캐릭터별 장비를 획득한다.
- 챕터 리셋과 무관하게 영구 유지되며 캐릭터 기본 스탯에 영향을 준다. (세부 스탯 미정)

## 패키지 게임 특성

| 요소 | 설계 방향 |
|------|-----------|
| 엔딩 구조 | 챕터 클리어 누적 → 최종 챕터 보스 격파 시 엔딩. 수집 요소에 따라 엔딩 분기 |
| 학습 곡선 | 초반은 낮은 카운트 적 + 낮은 숫자 카드 위주 → 후반 챕터에서 짧은 카운트 적 + 높은 숫자 카드 요구 |
| 전략 깊이 | 슬라이드 횟수 계산 + 핸드덱 우선순위 + 적 행동 예고 대응이 동시에 맞물림 |
| 재플레이 유인 | 덱 방향 (저숫자 다발 vs. 고숫자 필살) + 패시브 선택에 따라 완전히 다른 플레이 스타일 |
